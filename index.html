Drawing Equipment

The graphics tablets that I use are from the company, Wacom. The first model I recieved was an Intuos CTH-480, a standard graphics tablet suitable for beginners like myself at the time.

After completing highschool, I was generously given a Cintiq Pro 13HD, a graphics tablet with a "pen-on screen experience".

Anyhow, here's a full list of software and hardware I currently use when drawing:

  • Cintiq Pro 13HD
  • Acer Predator Helios 300 G3-571
  • Adobe Photoshop CC 2018
  • OBS Studio
